//we have used wamp server for our development

1. put the code under "www" directory of the web server running  

2. start the server 

3. http://<ip address>:port/simpliremind/newregister

4. login page would get open up.

5. Setup Google Cloud Project and obtain API-key and Project Id

6. To receieve notification from this server build apk from source file by providing SERVER_URL, and SENDER_ID (Google Project Id) in the android mobile device.

7.After Registration of mobile using .apk, Admin of cource can handle Notifications for particular users. 


8. test database is given as simpli_reminder.sql along with the code
